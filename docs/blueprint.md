# **App Name**: ChronoVault

## Core Features:

- Message Creation: Allow users to compose messages with text formatting, images, and attachments, and set a future date/time for delivery. Message can be public, private (one recipient), or group.
- Capsule Delivery: Provide a secure, access-controlled, and reliable means for capsule delivery at the defined future time. Capsules not accessed after 90 days expire.
- Location Suggester AI Tool: Summarize capsule contents to determine an appropriate place for opening it.
- Message Retrieval and Display: Offer a user-friendly interface to read the message upon capsule opening.
- Notifications: Implement email and/or SMS notifications to remind recipients when their time capsule is ready to be opened.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) evoking reliability and timelessness.
- Background color: Very light gray (#F5F5F5), nearly white, provides a clean backdrop to focus attention.
- Accent color: Muted violet (#9575CD), draws attention without clashing with other design elements.
- Font pairing: 'Literata' (serif) for body text, creating a readable and slightly vintage feel; 'Space Grotesk' (sans-serif) for headlines, offering a tech-forward and balanced contrast.
- Use simple, outlined icons to represent capsule functions.
- A clean, intuitive layout with a timeline to represent future delivery dates.
- Subtle animations when sending and opening time capsules, providing gentle feedback.
